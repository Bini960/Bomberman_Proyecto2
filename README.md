# Bomberman

Este repositorio contiene la implementación del clásico juego Bomberman para consola, desarrollado en C++. El proyecto aplica conceptos de programación concurrente mediante el uso de la biblioteca POSIX Pthreads. El sistema gestiona múltiples hilos independientes para controlar a los jugadores, los enemigos y las bombas de forma paralela. Además, integra mecanismos de sincronización para asegurar el manejo correcto de los recursos compartidos y la actualización fluida del entorno gráfico basado en ASCII-Art.
